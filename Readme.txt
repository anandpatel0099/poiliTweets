Run Code Information

First run this two files for collecting tweets and data preprocessing
1) For collecting data run tweet_collection.py
2) For data preprocessing run preProcess_tokenize.py

For finding prediction values for congress:
Use Congress folder to run below files
a) For finding features use features.py, CNG_train.txt as a dataset
b) For SVM use svm_code.py, CNG_train.txt and CNG_test.txt as a dataset
c) For wordcloud run wordcloud.py, CNG_test.txt as a dataset
d) For final result run generate_pie_chart.py, graph_CNG.csv

For finding prediction values for BJP:
Use BJP folder to run below files
a) For finding features use features.py, bjp_train.txt as a dataset
b) For SVM use svm_code.py, bjp_train.txt and BJP_test.txt as a dataset
c) For wordcloud run wordcloud.py, BJP_test.txt as a dataset
d) For final result run generate_pie_chart.py, graph_bjp.csv
